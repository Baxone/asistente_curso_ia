# Panel Kanban - Tareas

Aplicación Kanban en vanilla JavaScript con Tailwind CSS, drag and drop nativo HTML5 y persistencia en localStorage.

## Instalación

```bash
bun install
```

## Ejecución

```bash
bun start
```

Abre [http://localhost:3000](http://localhost:3000) en el navegador.

## Funcionalidades

- **3 columnas**: Inicio, En Proceso, Terminado
- **Formulario**: Crear tareas que se añaden a la columna Inicio
- **Drag and drop**: Arrastra las tarjetas entre columnas
- **Eliminar**: Botón de papelera en cada tarjeta
- **Persistencia**: Los datos se guardan en localStorage
