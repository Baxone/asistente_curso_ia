# Plan: Pomodoro Task
 
Quiero crear una aplicación en React con siguientes caracteristicas:

## Funcionalidad:

### Gestion de tareas:
- Crear una tarea.
- Editar una tarea.
- Eliminar una tarea.
- Marcar una tarea como completada.
- Cada tarea tiene que tener titulo, descripcion, prioridad (alta, media, baja)
- Ordenar las tareas por prioridad.
- Filtrar las tareas por estado

### Temporizador Pomodoro:
- Timer configurable por el usuario (por defecto 25 minutos, con 5 minutos de descanso)
- Ciclo: 4 pomodoros de trabajo, 1 descanso largo de 30 minutos.
- Asociar tarea activa al pomodoro actual.

### Estadisticas:
- Pomodoros completados.
- Tiempo total de trabajo.
- Tareas completadas.

### Restricciones:
- Componentes funcionales, no clases.
- Usa hooks.
