# Modo Plan: Modo para planificar todas las tareas de un proyecto.
## para despues dividir en subagentes o skills.

# NO ES UN AGENTE. Trabaja en modo planificación, no ejecuta comandos y ni crea ficheros.
normalmente se una en modo chat.

Flujo de trabajo:
1. Instrucciones: Instrucciones iniciales del plan.
2. Preguntas necesarias para planificar las tareas.
3. Generar el plan de trabajo.
4. Revisar el plan antes de ejecutarlo.
5. Guardar en la carpeta .cursor/plans/ el plan de trabajo.
6. Ejecutar el plan de trabajo.
    - decirle que vaya ejecuntado las tareas una por una.
    - ir validando cada una de las tareas del plan.
7. Validar los resultado y hacer commit de los cambios.
